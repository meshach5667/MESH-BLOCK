## Project Overview
This project is a simplified blockchain simulation developed in Python. It demonstrates the core principles of blockchain technology, including:

- Block creation
- Hashing with SHA-256
- Genesis block creation
- Proof of Work mining
- Chain validation
- Tampering detection

## Features
- Add new blocks
- Mine blocks using Proof of Work
- View the blockchain
- Validate blockchain integrity
- Tamper with a block for testing

## Project Structure
- `main.py` - main CLI program
- `blockchain/block.py` - Block class
- `blockchain/blockchain.py` - Blockchain logic
- `blockchain/utils.py` - helper functions
